My projects!

Here's some stuff I made

Spencer BMP Check it out on Github

SpencerBMP is a custom .BMP image parser written entirely in C. This allows you to either open existing or create new BMP files from C using an easy to understand & simple library. You are able to read/write the entire info header & pixel data of the image allowing for direct lowlevel control of all aspects of BMP files.

Linux Mint Password Generator Applet Check it out on the Linux Mint Website

As a long time user of Linux Mint, I created a configurable Cinnamon applet for generating passwords and copying them to your clipboard

ModBus TCP/IP A work project

I implemented the Modbus TCP/IP Protocol in C to work on custom PCB's that would read in data from a DB15 Serial and output it over Ethernet to be accessible as a PLC device. unfortunately as I wrote this for work I haven't uploaded the source code. Maybe eventually I will.
